# Guess Number

## Introduction

 guess an integer number from 1 to 100

## Installation
  
 1. execute below commands
 
    ```bash
    npm install
    ```

    ```bash
    npm install -g webpack
    ```  

    ```bash
    webpack
    ```  

    ```bash
    webpack-dev-server
    ```
  
  1. connect to http://localhost:8080/
